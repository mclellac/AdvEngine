# Managing Game Data

This guide explains how to manage the core data of your game, including global
state, characters, and database entries.

## Global State Editor

The Global State Editor is used to manage the global variables that control the
state of your game.

### Adding a New Variable

1.  Navigate to the **Global State** editor from the main sidebar.
2.  Click the **"Add New Variable"** button (the `+` icon) in the header bar.
3.  A new, empty row will be added to the list of variables.
4.  Edit the `ID`, `Type`, and `Value` of the new variable directly in the new
    row.

## Character Editor

The Character Editor is used to create and manage the characters in your game.

### Creating a New Character

1.  Navigate to the **Characters** editor from the main sidebar.
2.  Click the **"Add New Character"** button (the `+` icon) in the header bar.
3.  A new character with default values will be added to the list.
4.  Edit the character's properties (such as `ID`, `Portrait Asset`, etc.)
    directly in the list.

## Database Editor

The Database Editor is a collection of editors for managing the core data
objects of your game: Items, Attributes, and Verbs.

### Adding a New Entry

The process for adding a new entry is the same for each type of data:

1.  Navigate to the **Database** editor from the main sidebar.
2.  Select the tab for the type of data you want to edit (e.g., **Items**).
3.  Click the **"Add New Item"** (or **"Add New Attribute"**, etc.) button in
    the header bar.
4.  A new, empty row will be added to the table.
5.  Edit the properties of the new entry directly in the new row.
