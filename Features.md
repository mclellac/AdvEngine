# AdvEngine Features

This document provides a high-level overview of the features implemented in the AdvEngine application.

## Core Features

-   [x] **Project Management:** Create, open, and save projects.
-   [x] **Settings Management:** Application- and project-specific settings.
-   [x] **Game Launch:** Launch the game in Unreal Engine.
-   [x] **Localization:** Export and import user-facing text for translation.

## Editors

-   [x] **Scene Editor:** Create and edit game scenes, including hotspots.
-   [x] **Logic Editor:** A node-based editor for creating and editing game logic.
-   [x] **Dialogue Editor:** A tree-based editor for creating and editing dialogue.
-   [x] **Interaction Editor:** Define interactions between verbs and items.
-   [x] **Character Editor:** Manage game characters.
-   [x] **Quest Editor:** Create and edit quests and objectives.
-   [x] **Item Editor:** A database-style editor for in-game items.
-   [x] **Attribute Editor:** A database-style editor for character attributes.
-   [x] **Verb Editor:** A database-style editor for in-game verbs.
-   [x] **Global State Editor:** Manage global game variables.
-   [x] **Asset Manager:** Manage game assets, including images and animations.
-   [ ] **UI Layout Editor:** Design and lay out user interface elements.
-   [ ] **Font Editor:** Manage fonts used in the game.

## UI/UX

-   [x] **GNOME HIG Compliant:** The user interface follows the GNOME Human Interface Guidelines.
-   [x] **Libadwaita Widgets:** The UI is built using modern Libadwaita widgets.
-   [x] **Light and Dark Mode:** The application supports both light and dark modes.
-   [x] **Keyboard Shortcuts:** The application provides keyboard shortcuts for common actions.
