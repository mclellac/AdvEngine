# Preferences

The preferences window allows you to customize the AdvEngine application to your liking. You can access it by clicking the menu button in the top-right corner of the application window and selecting "Preferences."

## Appearance

*   **Application Theme**: Choose between the system's default theme, a light theme, or a dark theme.

## Editor

*   **Default Author**: Set the default author for new projects.
*   **Enable Autosave**: When enabled, your project will be automatically saved every 5 minutes.
*   **Default Node Width**: Set the default width for new nodes in the Logic Editor.
*   **Default Node Height**: Set the default height for new nodes in the Logic Editor.
*   **Enable Grid Snapping**: When enabled, nodes in the Logic Editor will snap to the grid when moved.
*   **Grid Size**: Set the size of the grid in the Logic Editor.

## External Tools

*   **Unreal Engine Editor Path**: Set the path to your Unreal Engine editor executable. This is required for the "Play" button to work.

## Project

*   **Project Name**: The name of your project.
*   **Default Starting Scene**: The scene that will be loaded when your game starts.
